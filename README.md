# 生日管家

一个离线可用的静态 Web App，用来管理公历和农历生日。

## 功能

- 添加、编辑、删除生日资料
- 支持公历生日和农历生日，包括闰月
- 计算近期生日和提前提醒日期
- 月历视图展示公历日期、农历日期和生日标记
- 公历转农历、农历转公历查询
- 为单个人或全部亲友导出未来 10 年 `.ics` 日历文件
- 为下一次生日生成 Google Calendar 添加链接
- 可选浏览器通知，页面打开时触发到期提醒

## 使用

直接用浏览器打开 `index.html`。数据保存在当前浏览器的 `localStorage` 中。

农历换算范围为 1900-01-31 至 2100-12-31。
